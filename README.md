# SiteAgentAI

AI Website Employee MVP demo for small service businesses. Captures leads from website visitors and shows business owners an AI-style leads dashboard.

## Tech stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- No database, auth, or backend (demo only)

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) for the landing page and live widget demo.

- **/** — Landing page with floating AI lead capture widget
- **/admin** — Sample leads dashboard (hardcoded demo data)

## Scripts

```bash
npm run dev    # Start development server
npm run build  # Production build
npm run start  # Start production server
npm run lint   # Run ESLint
```
